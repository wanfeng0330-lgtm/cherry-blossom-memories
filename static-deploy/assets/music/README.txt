# 樱花树时光机 - 背景音乐

将音乐文件放在此目录下。

建议的音乐格式：
- MP3
- M4A
- OGG

示例配置：
在 App.jsx 中的 DEMO_TRACKS 数组中添加音乐信息：

```javascript
const DEMO_TRACKS = [
  {
    id: 1,
    title: '樱花树下',
    artist: '歌手名称',
    url: '/assets/music/sakura.mp3'
  },
  {
    id: 2,
    title: '时光倒流',
    artist: '歌手名称',
    url: '/assets/music/time.mp3'
  }
];
```

推荐音乐风格：
- 温馨浪漫的钢琴曲
- 轻柔的吉他音乐
- 日系治愈系音乐
- 爵士乐
- 民谣
